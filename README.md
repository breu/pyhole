# pyhole - A modular IRC & Slack bot.

## Installation

````
git clone https://github.com/jk0/pyhole.git && cd pyhole
````

### Source

```
virtualenv venv
. venv/bin/activate

python setup.py flake8
python setup.py test
python setup.py develop

pyhole
````

### Docker

```
bash tools/docker_deploy.sh
```
